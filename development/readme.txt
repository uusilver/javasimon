If you use IDEA 12, please use this scheme and copy it into
Java Simon project. Thanks :-)

Move the file into your .IntelliJIdea12/config/codestyles directory, then:
- start IDEA,
- open the project,
- go to Settings,
- in the left section Project Settings choose Code Style,
- click the Manage button,
- select "Java Simon IDEA 12",
- click Copy to Project,
- confirm, close... you're ready to go.
