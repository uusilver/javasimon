Java Simon Demo Web Application

Java Simon provides a few ways how to monitor your web-requests out of the box:

  • Simon Serlvet Filter - is mandatory and perfoms the measuring itself. It comes with simple plain text "console" - check it here (or "help" page)
  • Embedded Console comes in two ways - as a filter or as a servlet. Functionally they are the same, but filter is easier to set-up so it is not monitored - just put it before the Simon Servlet Filter.
